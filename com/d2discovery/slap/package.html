SLAP, Semantically Linked Association Prediction

Original authors: Bin Chen, Qian Zhu

This package has been adapted and organized from the code developed at the Cheminformatics and Chemogenomics Research Group, led by David Wild, at Indiana University. Data2Discovery Inc. has been founded by David Wild and Ying Ding to sustain, extend, distribute and commercialize SLAP and related methods.

Some aims of this newly organized package:

  • Encapsulate code for convenient deployment via J2EE web app (via WAR).
  • Employ context and servlet configuration parameters (via web.xml) to removed hard-coded hostname, db-name, etc.

Note that this package accesses the Chem2Bio2RDF PostgreSQL database directly via JDBC (normally port 5432), and the SLAP REST API (normally port 80).
